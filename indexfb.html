¿QUÉ ES UNA FUNCIÓN?

Una función es un grupo de instrucciones (líneas de código) que puedes guardar con un nombre.
Después, puedes usar ese nombre para ejecutar todas esas instrucciones cuando quieras.

Piensa en una cafetera: aprietas el botón, la cafetera hace café. No tienes que hacer todo tú (calentar el agua, filtrar el café, etc.), la cafetera ya sabe cómo hacerlo.

En programación, una función funciona igual:
le das un nombre,
le enseñas qué debe hacer,
y luego la llamas para que lo haga.

¿CÓMO SE CREA UNA FUNCIÓN EN JAVASCRIPT?

function saludar() { console.log("Hola");}

function: indica que estamos creando una función.
saludar: nombre de la función.
( ): paréntesis para recibir datos (parámetros).
{ }: llaves con el código a ejecutar.

La palabra “declarar” en programación significa anunciar o decirle al programa que algo existe. Es como decirle a JavaScript: “Voy a crear algo con este nombre, guárdalo porque luego lo voy a usar”.
Cuando declaras una variable, le estás diciendo al programa que existe un espacio en memoria con ese nombre.


let nombre="Anna";

Aquí estás declarando una variable llamada nombre y asignándole el valor "Ana". Javascript ahora sabe que existe una variable llamada nombre y que contiene el valor "Ana".

¿QUE HACE JAVASCRIPT CON ESTO?


- Crea un espacio en la memoria del computador.
- Le pone una etiqueta (el nombre de la variable nombre).
- Guarda el valor "Ana" dentro de ese espacio.

Para que la función realmente haga algo, debes invocarla o llamarla.
Eso significa: decirle a JavaScript que ejecute el codigo dentro de esa función.

¿COMO SE INVOCA UNA FUNCIÓN?

nombreDeLaFuncion( ) ;
() los paréntesis indican que quieres ejecutar la función.
Incluso si la función no recibe datos, los paréntesis son obligatorios para ejecutarla.

EJEMPLO:
saludar( ) { console.log("¡Hola!"); }

Invocar la función:
saludar( ) ;



INVOCACIÓN CON PARÁMETROS:

Si la función necesita datos, los pones dentro de los paréntesis:

saludarPersona(nombre) {console.log("Hola, " + nombre + "!") ; }

saludarPersona ("Ana"); // Hola, Ana!
saludarPersona ("Carlos"); // Hola, Carlos!

"Ana" y "Carlos" son argumentos.
nombre es el parámetro dentro de la función.

Cuando creas una función, puede tener nombre o no tener nombre.
Si tiene nombre, función nominada (o nombrada). Si no tiene nombre es, función anónima.

NOMINADA:

function saludarNom() { console.log("Hola (nominada)"); }

¿Qué pasa cuando tiene nombre?


El programa puede:

- reconocerla
- guardarla con ese nombre
- llamarla más adelante

Se invoca así: saludar( );



ANÓNIMA:


function() {console.log("Hola (anónima)");};


Como no tiene nombre, no puedes llamarla así: ???( ) ;
no existe un nombre para escribir aquí

¿Cómo se usa una función anónima?


La guardas dentro de una variable:
const saludar = function( ) {console.log("Hola!");};

ahora si se puede usar:
saludar( ) ;

Los parámetros son variables que una función espera recibir para poder trabajar. Sirven para que la función pueda usar información que tú le envías desde afuera. La función por sí sola no sabe a quién saludar, qué sumar, qué mostrar… Así que necesita recibir datos.

¿CÓMO SE USA EL PARAMETRO?

function saludar(nombre)

Declaras la función con un parámetro (una variable vacía):

console.log("Hola, " + nombre);

Dentro de la función, usas el parámetro como una variable normal.

saludar("Ana");

Ese dato reemplaza al parámetro cuando llamas a la función.

Y la funcion imprime:

RETURN

function sumar(a, b) {
    return a + b;
}

let resultado = sumar(3, 4);
console.log(resultado); // 7

Return devuelve un valor desde la función para usarlo más adelante.

FUNCIONES FLECHA (ARROW FUNCTIONS)

// Función clásica
function saludar(nombre) {
    return 'Hola ' + nombre;
}

// Arrow function
const saludarFlecha = (nombre) => 'Hola ' + nombre;

console.log(saludarFlecha('Pedro'));
