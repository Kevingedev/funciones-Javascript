Funciones Básicas de JavaScript

¿QUÉ ES UNA FUNCIÓN?

Una función es un grupo de instrucciones (líneas de código) que puedes guardar con un nombre.
Después, puedes usar ese nombre para ejecutar todas esas instrucciones cuando quieras.

Piensa en una cafetera: aprietas el botón, la cafetera hace café. No tienes que hacer todo tú (calentar el agua, filtrar el café, etc.), la cafetera ya sabe cómo hacerlo.

En programación, una función funciona igual:
le das un nombre,
le enseñas qué debe hacer,
y luego la llamas para que lo haga.

¿CÓMO SE CREA UNA FUNCIÓN EN JAVASCRIPT?

function saludar() {
    console.log("Hola");
}

function: indica que estamos creando una función.
saludar: nombre de la función.
( ): paréntesis para recibir datos (parámetros).
{ }: llaves con el código a ejecutar.

INVOCACIÓN / LLAMADA

saludar();

Los paréntesis son obligatorios incluso si la función no recibe datos.

PARÁMETROS Y ARGUMENTOS

function saludarPersona(nombre) {
    console.log("Hola, " + nombre + "!");
}

saludarPersona("Ana");
saludarPersona("Carlos");

nombre es un parámetro.
Ana y Carlos son argumentos.

RETURn

function sumar(a, b) {
    return a + b;
}

let resultado = sumar(3, 4);
console.log(resultado); // 7

Return devuelve un valor desde la función para usarlo más adelante.

FUNCIONES NOMINADAS Y ANÓNIMAS

// Nominada
function saludarNom() {
    console.log('Hola (nominada)');
}

// Anónima
const saludarAnon = function() {
    console.log('Hola (anónima)');
};

FUNCIONES FLECHA (ARROW FUNCTIONS)

// Función clásica
function saludar(nombre) {
    return 'Hola ' + nombre;
}

// Arrow function
const saludarFlecha = (nombre) => 'Hola ' + nombre;

console.log(saludarFlecha('Pedro'));
