Esta función está ahí para cuando necesitamos hacer algo con cada elemento de un array. Es simple, directo, y hace exactamente lo que dice: "por cada elemento, haz esto". 

                
                    
array.forEach(function(elemento, índice, array) {
    // código a ejecutar para cada elemento
});

  • elemento: El valor actual del array que está siendo procesado.
  • índice: La posición del elemento dentro del array.
  • array: Referencia al array sobre el que se llamó a forEach.



La función orgánica map transforma los datos y devuelve siempre un array. "Por cada elemento que entra, ejecuta una acción, y el resultado de esa acción es el nuevo elemento que sale."

  • No modifica el original: Es crucial entender que map() siempre devuelve un array completamente nuevo. El array original permanece inalterado.
  • Mismo número de elementos: El array nuevo que resulta siempre tendrá exactamente el mismo número de elementos que el array original.


                
                    
const nuevoArray = array.map(function(elemento) {
    // código a ejecutar para cada elemento
});

Nota: La función debe retornar algo. Si no, vas a terminar con un array lleno de undefined.



El método filter() opera bajo los siguientes principios:

  • Iteración: Recorre cada elemento del array original, uno por uno, en orden ascendente (indice más0 bajo al más alto).
  • Ejecución de la Callback: Por cada elemento, ejecuta una función de callback que le proporcionas.
  • Evaluación de la Condición: La función callback debe devolver un valor booleano (true o false). Esta es la condición de filtrado.
  • Si la callback devuelve true para un elemento, ese elemento se incluye en el nuevo array.
  • Si la callback devuelve false para un elemento, ese elemento se omite del nuevo array.
  • Inmutabilidad: filter() nunca modifica el array original. Siempre devuelve un nuevo array con los elementos filtrados. Si ningún elemento cumple la condición, devuelve un array vacío ([]).


                
                    
const nuevoArray = array.filter(callback(elemento, indice, array));
